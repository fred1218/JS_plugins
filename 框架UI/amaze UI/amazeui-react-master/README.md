## 开发及构建

### 目录结构

```
├── package.json
├── dist          # 构建目录
├── docs          # 文档及测试示例
├── src           # 组件源文件（JSX）
└── utils
```

### 开发及文档

```
gulp dev
```

### 构建

```
gulp build
```
